# Medical Store Management System

A Java-based desktop application developed to manage and streamline the daily operations of a medical store or pharmacy. This system helps in handling medicine inventory, sales management, customer details, billing, supplier management, and user authentication efficiently.

## 📌 Project Overview

The Medical Store Management System is designed to automate the manual processes involved in managing a pharmacy. It reduces paperwork, improves accuracy, saves time, and provides better control over medicine stock and sales transactions.

This application is suitable for:
- Medical Stores
- Pharmacies
- Small Healthcare Shops
- Medicine Inventory Management

---

## 🚀 Features

- User Login Authentication
- Medicine Inventory Management
- Add New Medicines
- Update Medicine Details
- Delete Medicines
- Search Medicines
- Category Management
- Supplier Management
- Customer Management
- Billing and Invoice Generation
- Sales Management
- Stock Availability Tracking
- Password Change Option
- User-Friendly Graphical Interface

---

## 🛠️ Technologies Used

- **Programming Language:** Java
- **GUI Framework:** Java Swing
- **IDE:** NetBeans / Eclipse
- **Database:** MySQL
- **JDBC:** Java Database Connectivity
- **Version Control:** Git & GitHub
